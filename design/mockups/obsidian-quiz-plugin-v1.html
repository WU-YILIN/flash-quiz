Question 07 · Weak Memory Boost
在 Retrieval-Augmented Generation 流程中，检索阶段的核心价值是什么？
A
让模型输出更长的回答，以便覆盖更多知识点。
B
在生成前补充外部上下文，降低模型仅依赖参数记忆回答的风险。
C
把所有笔记压缩成固定向量后，完全替代语言模型的推理过程。
D
直接根据文档标题决定答案，从而减少检索延迟。

回答错误，正确答案是 B

检索的作用是在生成前把和当前问题最相关的文档片段送入上下文窗口。这样模型不是“凭印象回答”，而是依据外部知识回答，因此更适合基于 Obsidian 笔记做 quiz 和记忆巩固。
